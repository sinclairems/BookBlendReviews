# Back-end Requirements

## Use MySQL and the Sequelize ORM for the database

* sketch out the database

* make a schema

* make seeds

* test database

## Use Node.js and Express.js to create a RESTful API

* use node and express to make a server

* use express to make a router

* use express to make routes 

* test the routes with Insomnia

## Have both GET and POST routes for retrieving and adding new data

* make a get route

* make a post route

* test the routes with Insomnia

## Include authentication (express-session and cookies)

* cookies for sessions

* JSON web tokens for authentication 

# Front-end Requirements

## Use Handlebars.js as the template engine

* use the [express-handlebars](https://www.npmjs.com/package/express-handlebars) package for Views

## Be deployed using Heroku (with data)

* use the [Guide to Deploy with Heroku and MySQL](https://coding-boot-camp.github.io/full-stack/heroku/deploy-with-heroku-and-mysql)
